Contributing

How to get involved in the PyScript Collective.

Viewer

A person who shows up at this website and clicks on the examples in the Gallery.

Coder

A person who uses pipx to run the examples locally.

Contributor

A person makes a Python project, installs pyga as a dependency, and starts making examples.

Collaborator

The core Collective team that takes long-term ownership of all the examples, keeping the tests running, and keeping the code up-to-date.
